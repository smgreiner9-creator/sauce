# Changelog

## v1.0.0 — Initial Release

- T-Pain style hard-tune pitch correction (0ms retune speed)
- YIN pitch detection (80Hz–1kHz vocal range)
- TD-PSOLA pitch shifting
- Cepstral formant shift (-12 to +12 semitones, bypassed at 0)
- Key selection (C through B) + Scale (Chromatic, Major, Minor)
- Dry/Wet mix and Input/Output gain controls
- Neon cyberpunk GUI with 3D "SAUCE" title
- Real-time pitch meter (detected → target)
- Windows VST3 installer via GitHub Releases
